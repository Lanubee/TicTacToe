
PROJECT TITLE: TicTacToe  
PURPOSE OF PROJECT: University Project
VERSION or DATE: March 2022
HOW TO START THIS PROJECT: Run the code on your IDE
AUTHORS: Adam Ezzeldin

The game has a basic UI where you need to enter a personal username to start. The game has features like to reset the game and to start the game and stops when a winning condition or draw condition is met. The game is single player only and includes a basic AI for the player to play against.
